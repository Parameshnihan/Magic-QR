Build a production-ready SaaS web application called “ReviewFlow Pro” — a premium reputation management and Google review automation platform for agencies, hotels, restaurants, healthcare brands, and multi-location businesses.

The platform helps agencies onboard clients, generate QR codes, collect customer ratings, redirect positive reviews to Google Business Profile, capture negative feedback privately, automate notifications, manage billing/subscriptions, and provide analytics dashboards for admins, managers, and clients.

====================================================
RATING FLOW LOGIC
====================================================

If rating = 1,2,3:
- Show feedback form
- Optional complaint details
- Optional contact field

After submission:
- Store feedback privately
- Send notification to:
  - Client WhatsApp
  - Client email
  - Admin
  - Manager

Then show:
- Thank you animation
- “We appreciate your feedback”

Do NOT redirect to Google.

If rating = 4 or 5:
- Show recommended keyword chips
- Show AI-generated review suggestions
- Optional custom review textarea

Then:
- Submit review data
- Redirect to Google review page
- Auto-copy review text
- Open Google review URL

Important:
Implement the strongest possible Google review flow allowed within Google policies.

If direct API posting is not allowed by Google, implement the closest seamless UX possible:
- pre-filled review text
- auto copy
- one-click redirect
